# דף נחיתה אישי — הודיה חדד

דף נחיתה בפורמט HTML סטטי לפורטפוליו מקצועי של עובדת סוציאלית המתמחה בילדים עם צרכים מיוחדים באומנה.

---

## מה זה?

קובץ HTML יחיד שמהווה דף נחיתה מקצועי, הכולל:

- **Hero** — תמונת פרופיל, שם ותואר
- **אודותי** — תיאור אישי ו-3 כרטיסיות: ניסיון, התמחות, ערכים
- **מה אני מציעה** — רשימת שירותים מקצועיים
- **צרי קשר** — קישורים ישירים למייל, WhatsApp, פייסבוק ואינסטגרם

הדף בנוי בעברית מלאה (RTL), עם עיצוב חם ואנושי המתאים לתחום העבודה הסוציאלית.

---

## איך להריץ?

אין צורך בשרת, בהתקנות, או בקוד נוסף.

1. פתחי את סייר הקבצים
2. לחצי **לחיצה ימנית** על `hodaya_portfolio.html`
3. בחרי **"פתח עם"** ← Chrome / Edge / Firefox

זהו. הדף נפתח ישירות בדפדפן.

---

## להוספת תמונת פרופיל

1. שמרי קובץ תמונה בשם `photo.jpg` באותה תיקייה כמו קובץ ה-HTML
2. רעננ את הדפדפן — התמונה תופיע אוטומטית

---

## מה למדתי בתהליך

- **HTML + CSS בסיסי** — מבנה דף, תגיות, קישורים
- **RTL** — כיצד לבנות דף עברי נכון עם `dir="rtl"`
- **עיצוב רספונסיבי** — שימוש ב-CSS Grid כדי שהדף ייראה טוב גם בפלאפון
- **קישורים חכמים** — `mailto:` לפתיחת מייל, `wa.me/` לפתיחת WhatsApp ישירות
- **שימוש ב-Claude Code** — כיצד לתאר מה שאני רוצה ולקבל קוד עובד, לשנות פרטים (כמו מייל וקישורים), ולבצע שיפורי עיצוב בשלבים
