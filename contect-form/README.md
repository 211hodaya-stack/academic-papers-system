# טופס יצירת קשר

טופס יצירת קשר סטטי בעברית עם ולידציה בצד הלקוח, כתוב ב-HTML, CSS ו-JavaScript טהור — ללא ספריות חיצוניות.

---

## איך להריץ

אין צורך בהתקנות. פשוט פתח את הקובץ בדפדפן:

```
index.html → לחץ פעמיים או פתח דרך Live Server ב-VS Code
```

---

## אימותים

| שדה | חוק |
|-----|-----|
| שם מלא | לא ריק |
| אימייל | פורמט תקני — חייב להכיל `@` ונקודה |
| טלפון | 9–10 ספרות בלבד, ללא אותיות או תווים מיוחדים |
| הודעה | לפחות 10 תווים |

כאשר שדה לא תקין — מסגרת אדומה + רקע אדמדם + הודעת שגיאה מתחת לשדה.  
כאשר כל השדות תקינים — alert עם הנתונים שהוזנו, ניקוי הטופס, והצגת הודעת הצלחה למשך 3 שניות עם אנימציית fade.

---

## מה למדתי

- **ולידציה ב-JavaScript** — שימוש ב-`addEventListener('submit')` ו-`preventDefault()` לבדיקת שדות לפני שליחה
- **Regex** — כתיבת ביטויים רגולריים לבדיקת פורמט אימייל (`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`) וטלפון (`/^\d{9,10}$/`)
- **מניפולציה של DOM** — הוספה והסרה דינמית של classes לשינוי עיצוב (`classList.add/remove`)
- **אנימציות CSS** — שימוש ב-`@keyframes` לאנימציות `fadeIn` ו-`fadeOut`, ושליטה בהן מ-JavaScript
- **RTL ועיצוב עברי** — שימוש ב-`dir="rtl"` ו-`lang="he"` לפריסה נכונה של ממשק בעברית
- **עיצוב עם CSS Variables** — הגדרת צבעים מרכזיים ב-`:root` לשימוש חוזר עקבי
